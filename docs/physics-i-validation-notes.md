# Validación de Física I

La asignatura debe superar:

- validación JSON de la arquitectura y las seis unidades;
- auditoría semestral estricta;
- comprobación de renderizadores y notación MathJax;
- generación completa del sitio;
- validación de enlaces internos.

No se considera revisión académica humana concluida. El estado interno permanece en `review`.
