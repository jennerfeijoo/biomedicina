Bioinformática

Análisis computacional de datos biológicos y biomédicos para interpretar secuencias, anotaciones, bases de datos y resultados ómicos.

Ver todas las asignaturas Inicio ← Biofotónica Bioinstrumentación

Propósito formativo

La bioinformática conecta biología molecular, datos y computación para estudiar genes, proteínas, variantes, expresión génica y relaciones funcionales. Es esencial en investigación biomédica moderna, diagnóstico molecular, medicina personalizada, vigilancia genómica y análisis de grandes volúmenes de datos biológicos.

Objetivos de aprendizaje

  • Comprender qué tipos de datos biológicos se analizan mediante bioinformática.
  • Identificar el papel de secuencias, alineamientos, anotaciones y bases de datos biológicas.
  • Relacionar algoritmos bioinformáticos con preguntas biomédicas concretas.
  • Interpretar resultados computacionales considerando calidad de datos, contexto biológico y límites metodológicos.

Módulos sugeridos

  • Datos biológicos digitales: secuencias, genes, proteínas y anotaciones.
  • Bases de datos biológicas y recuperación de información.
  • Alineamiento de secuencias y búsqueda de similitud.
  • Introducción a genómica, transcriptómica y análisis ómico.
  • Pipelines reproducibles y control de calidad.
  • Interpretación biomédica de resultados bioinformáticos.

Conceptos clave

  • secuencia
  • alineamiento
  • anotación funcional
  • base de datos biológica
  • homología
  • genoma
  • transcriptoma
  • pipeline
  • control de calidad
  • reproducibilidad
